# RideShare Optimizer

A sophisticated route optimization and platform management system. This project features machine learning-inspired algorithms for efficient ride-sharing, carbon savings analytics, and a comprehensive administration suite.

## 🚀 Key Features

- **Google Authentication**: Secure sign-in integrated with Firebase.
- **Route Optimization**: Real-time comparisons of DP, Nearest Neighbor, and Cheapest Insertion algorithms.
- **Carbon Analytics**: Comprehensive tracking of CO2 savings and trip efficiency.
- **Admin Dashboard**:
  - **Platform Analytics**: Live demand heatmap and performance metrics.
  - **User Management**: Real-time role control (Promote to Driver / Demote to Rider).
- **Responsive Design**: Modern, premium UI built with React, Tailwind CSS, and Framer Motion.

## 🛠️ Tech Stack

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, Lucide Icons, Shadcn UI.
- **Backend**: Python, Flask, SQLAlchemy, JWT Authentication, Firebase Admin SDK.
- **Database**: SQLite (SQLAlchemy).

## 🔒 Security

This project uses modern security practices, including:
- JWT-based authorization for all protected simulator and admin routes.
- Firebase integration for verified identity management.
- Segregated environment secrets protection.

## 📈 Administration

The system includes a dedicated Administration suite with separate views for high-level platform analytics and granular user management.
